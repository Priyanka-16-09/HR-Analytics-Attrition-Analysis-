|| HR Analytics (Attrition Analysis) ||

📌 1. Project Overview:-

This project focuses on analyzing employee attrition data to understand why employees leave the organization.
The aim is to identify patterns based on salary, department, experience, and job roles.

For this project, I used SQL for data analysis, Excel for basic exploration, and Power BI to build an interactive dashboard.

🎯 2. Objective:-

* Analyze employee attrition trends
* Identify key factors influencing employee turnover
* Build a dashboard to support decision-making


3. Tools & Technologies Used:-

* MySQL (Data analysis & querying)
* Power BI (Dashboard & visualization)
* Excel (Data exploration & validation)

4. Dataset Information:-
The dataset contains employee-level details such as:

* Age
* Department
* Job Role
* Monthly Income
* Years at Company
* Attrition (Yes/No)

5. Work Done in SQL:-

I used SQL to explore and analyze the dataset before visualization.
Key tasks performed:

* Checked total number of employees
* Calculated attrition count and attrition rate
* Analyzed attrition by department
* Grouped employees based on salary levels (Low, Medium, High)
* Evaluated attrition based on years at company
* Compared attrition across job roles

These queries helped in understanding patterns before building the dashboard.

6. Work Done in Excel:-

Excel was used for quick analysis and validation.
Key tasks:

* Created pivot tables to summarize attrition data
* Compared attrition across departments and salary groups
* Performed initial data checks (missing values, structure)

This step helped in validating insights before moving to Power BI.

 7. Dashboard Features (Power BI)

 KPI Cards

*Total Employees
*Attrition Count
*Attrition Rate

Visualizations Used

1. Attrition by Years at Company

* Chart Type: Column Chart
* Shows how attrition varies with experience
* Highlights early-stage attrition

2. Attrition by Department

* Chart Type: Bar Chart
* Compares attrition across departments
* Helps identify high attrition areas

3. Attrition by Salary Group

* Chart Type: Column Chart
* Groups employees into Low, Medium, High salary
* Shows impact of salary on attrition

 4. Attrition by Job Role

* Chart Type: Donut Chart
* Shows contribution of each role to total attrition

5. Filters (Slicers)

* Gender
* Job Role

These filters allow interactive analysis.



8. Key Insights:-

* Employees with lower salary levels have higher attrition
* Most employees leave within the first few years
* Research & Development and Sales departments show higher attrition
* Some job roles contribute more to employee turnover

9.  Project Workflow:-

1. Data explored and analyzed using SQL
2. Initial validation done in Excel using pivot tables
3. Data connected to Power BI
4. Measures created (Attrition Count, Rate)
5. Dashboard designed with KPIs and charts
6. Insights derived from analysis

📌 10. Conclusion

This project provides a clear understanding of employee attrition patterns and highlights key areas where organizations can focus to improve retention.
